# v1.0.0 GitHub 릴리스 생성 디자인

## 개요

mcp-server-steam 프로젝트의 첫 번째 공식 릴리스(v1.0.0)를 GitHub에 생성합니다.

## 릴리스 절차

### 1. Git 태그 생성 및 푸시

```bash
git tag -a v1.0.0 -m "Release v1.0.0"
git push origin v1.0.0
```

### 2. 소스 코드 압축 파일 생성

git archive를 사용하여 태그 기준으로 소스 코드를 압축합니다.

```bash
git archive --format=tar.gz v1.0.0 > mcp-server-steam-v1.0.0.tar.gz
git archive --format=zip v1.0.0 -o mcp-server-steam-v1.0.0.zip
```

### 3. 릴리스 노트 작성

상세 한국어 릴리스 노트를 작성합니다. 다음 섹션들을 포함:

- 헤더: 릴리스 버전과 날짜
- 개요: MCP 서버 소개 및 Steam Web API 통합 설명
- 주요 기능: 3가지 카테고리 (사용자 프로필, 게임 정보, 커뮤니티)
- 설치 방법: 단계별 가이드
- 사용 방법: Claude Desktop 설정 예시
- 사용 가능한 도구: 모든 MCP 도구 목록
- 주요 변경사항: v1.0.0 기준 구현된 기능들
- 요구 사항: Python 버전, FastMCP 버전
- 참고 링크: Steam API 문서, 이슈 트래커

### 4. GitHub 릴리스 생성

gh CLI를 사용하여 릴리스를 생성합니다.

```bash
gh release create v1.0.0 \
  --title "v1.0.0: 첫 번째 공식 릴리스" \
  --notes-file RELEASE_NOTES.md \
  mcp-server-steam-v1.0.0.tar.gz \
  mcp-server-steam-v1.0.0.zip
```
